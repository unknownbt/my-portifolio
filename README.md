# 3D Live Portfolio

A visually stunning, high-performance 3D portfolio website built with Vanilla JS, HTML, CSS, and Three.js. It features gyroscopic wireframe models, particle effects, web audio synthesizer sound effects, and responsive glassmorphic cards.

## 🔗 Live Demo
Check out the live website deployed on Vercel:
**[https://my-portifolio-iota-woad.vercel.app](https://my-portifolio-iota-woad.vercel.app)**

## 🚀 Features
- **Gyroscopic 3D Globe**: Double-shell rotating sphere with particle nodes and satellite packet travelers representing full-stack API connections.
- **Interactive Particle Bursts**: Click on the background to deploy explosive particle data nodes.
- **Web Audio Synth**: Dynamic audio feedback using Web Audio API oscillators, including hover tick, packet click beeps, and success sweep chime.
- **HUD Telemetry Bar**: Shows real-time simulated telemetry log items and calculated frames per second (FPS).
- **Responsive Design**: Elegant cyberpunk grid overlay and modern typography fully optimized for mobile, tablet, and desktop layouts.

## 🛠️ Tech Stack
- **Core**: HTML5, Vanilla JavaScript (ES6+), CSS3
- **3D Graphics**: Three.js, OrbitControls
- **Icons**: Font Awesome
- **Deployment**: Vercel

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/unknownbt/my-portifolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the dev server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
